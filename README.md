## Overview

Tmmni is an AI-powered healthcare web application designed to assist users in assessing potential skin conditions from uploaded images.

The platform provides early-stage health insights and encourages users to seek professional medical consultation when needed.

## Features

- AI-based skin condition analysis
- Image upload and processing
- Early-stage health insights
- Support for informed medical decisions
- Connection with healthcare professionals
